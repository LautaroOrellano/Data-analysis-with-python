# Data Analysis Projects in Python

Este repositorio contiene un portafolio de proyectos de análisis de datos y visualización desarrollados en Python, utilizando datasets reales de distintas fuentes y dominios. 
Cada proyecto está diseñado para extraer insights accionables y demostrar habilidades en análisis exploratorio de datos (EDA), visualización, limpieza de datos e interpretación de resultados.

## Estructura del repositorio

Cada proyecto tiene su propia carpeta, que incluye:

- `data/raw/` – Datasets originales sin modificar.
- `data/processed/` – Datos procesados listos para análisis (si aplica).
- `notebooks/` – Notebooks de Jupyter con análisis, visualizaciones y markdown explicativo.
- `requirements.txt` – Dependencias necesarias para reproducir el proyecto.
- `docs/` – Documentación adicional, si aplica.

## Proyectos incluidos (ejemplos)

- **saas-churn-analysis/** – Análisis de churn de clientes en un servicio SaaS.
- **telco-customer-churn/** – Análisis de cancelación de clientes de telecomunicaciones.
- **marketplace-pricing-analysis/** – Evaluación de precios y comportamiento de productos en un marketplace.
- **data-quality-missing-values/** – Proyectos de análisis de calidad de datos y tratamiento de valores faltantes.
